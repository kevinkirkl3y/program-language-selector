, you should learn Ruby! Ruby is an object oriented programming language developed to carry a careful balance. Blended from creator, Yukihiro "Matz" Matsumoto's favorite languages (Perl, Smalltalk, Eiffel, Ada and List) Ruby formed a new language that balanced functional programing with imperative programming.

"...trying to make Ruby natural, not simple..."

Yukihiro "Matz" Matsumoto

, you should learn C#(C-Sharp)! C# is an object oriented programming language intended to be general purpose, simple and modern. It provides strong type checking, array bounds checking, detects uninitialized variables and uses automatic garbage collection. It focuses on maintaining portability and provides support for internationalization. Applications created using C# are intended to be economical with regard to processing power and memory requirements.

"I value simplicity over everything; I always look for simplicity."

Anders Hejlsberg (C# Creator)

, you should learn Python! Python can be easy to pick up either as a first time programmer or after gaining experience with other languages. It is an interpreted object oriented, high-level programming language with dynamnic semantics. Featuring dynamic typing and binding it has become attractive for rapid application development as well as for stitching existing components together. It is open source and available to be freely distributed.

"Python is an experiment in how much freedom programmers need. Too much freedom and nobody can read another's code; too little and expressiveness is endangered."

Guido Van Rossum (Python Creator)
